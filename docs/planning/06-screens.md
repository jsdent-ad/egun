## 게시판 페이지 (원페이지 + 앵커 네비게이션)

### B-01: 이건치과소개
- ID: board-about
- 경로: /about
- 서브메뉴: 진료철학 | 의료진소개 | 진료일정 | 내부전경 | 디지털기공소 | 오시는길
- 섹션:
  - A. 진료철학: 텍스트 → 원장님 슬라이드 사진 (이재성, 유수현 교대)
  - B. 의료진소개: 약력 + 손글씨 편지 사진 + 면허증 사진 + 진료 사진 (서울권치과 참고)
  - C. 진료일정: 캘린더 (본관 일반 / 별관 소아치과 / 별관 교정과)
  - D. 내부전경: 본관 + 별관 사진 갤러리
  - E. 디지털 기공소: 영상 + 사진 + 구성원 소개
  - F. 오시는길: 네이버 지도 (플레이스 연동) + 진료시간표 + 주차장/대중교통 실사진
- 컴포넌트: AnchorNav, DoctorProfile, ClinicCalendar, MapEmbed, PhotoGallery

- 컴포넌트: TreatmentSection, BenefitList, CaseGallery, FaqAccordion, BlogLink, CtaSection

## 관리자 페이지

### A-01: 관리자 로그인
- ID: admin-login
- 경로: /admin/login
- 구성: 이메일/비밀번호 입력 + 로그인 버튼
- 인증: Supabase Auth

### A-02: 관리자 대시보드
- ID: admin-dashboard
- 경로: /admin
- 구성: 메뉴 카드 (상담DB, 증례관리, 공지사항)
- 통계: 미확인 상담 건수 표시

### A-03: 상담 DB 관리
- ID: admin-consultations
- 경로: /admin/consultations
- 구성: 테이블 (이름, 연락처, 신청일, 상태, 메모)
- 기능: 상태 변경 (pending→contacted→completed), 메모 추가, 삭제

### A-04: 증례 관리
- ID: admin-cases
- 경로: /admin/cases
- 구성: 게시판별 필터 + 증례 목록 + 추가/수정/삭제
- 기능: 사진 업로드 (before/after), 제목/설명 편집, 블로그 링크 추가, 정렬 순서 변경

### A-05: 공지사항 관리
- ID: admin-notices
- 경로: /admin/notices
- 구성: 공지 목록 + 추가/수정/삭제
- 기능: 제목, 내용, 날짜, 이미지 업로드, 활성/비활성 토글
