# 서울이건치과 홈페이지 PRD

## MVP 캡슐
1. 목표: 서울대 출신 원장 2인의 책임진료 철학을 전달하고, 온라인 상담 전환을 극대화하는 치과 홈페이지
2. 페르소나: 수원 영통구 인근 30~60대 치과 환자 (임플란트, 교정, 소아치과 니즈)
3. 핵심 기능: FEAT-1 진료 정보 제공, FEAT-2 온라인 상담 수집, FEAT-3 관리자 콘텐츠 관리
4. 성공 지표 (노스스타): 월간 상담 신청 건수
5. 입력 지표: 페이지 체류시간, 상담바 클릭률
6. 비기능 요구: 모바일 반응형, 3초 이내 초기 로딩
7. Out-of-scope: 온라인 결제, 진료 예약 캘린더 연동, 회원가입
8. Top 리스크: 증례사진/콘텐츠 수급 일정
9. 완화/실험: 증례사진 자리에 플레이스홀더 배치, 관리자에서 추후 업로드
10. 다음 단계: /screen-spec 실행

## 2. 핵심 기능

### FEAT-1: 진료 정보 제공 (7개 게시판)
- **목적**: 치과의 전문성과 진료 분야를 체계적으로 전달
- **Why**: 환자가 내원 전 충분한 정보를 얻고 신뢰를 형성
- **우선순위**: MVP 필수

| 게시판 | 서브 항목 |
|--------|----------|
| 1. 이건치과소개 | 진료철학, 의료진소개, 진료일정(캘린더), 내부전경, 디지털기공소, 오시는길 |
| 2. 자연치아살리기 | 충치치료, VPT 신경보존술, 근관치료, 잇몸치료 |
| 3. 임플란트 | 올온, 즉시로딩, 네비게이션, 전치부, 자가혈, 상악동거상술, 시니어, 당뇨 |
| 4. 심미보철 | 최소삭제 라미네이트, 앞니 레진빌드업, 잇몸성형술 |
| 5. 서울이건 교정치료 | 인비절라인 투명교정, 소아성장기교정 |
| 6. 소아치과 치료 | 소아충치치료, 웃음가스치료 |
| 7. 이건 미디어 | 카톡상담, 빠른상담, 블로그, 유튜브, 가는길 |

### FEAT-2: 온라인 상담 수집
- **목적**: 환자 DB를 수집하여 상담 전환율 극대화
- **Why**: 24시간 상담 접수 가능, 잠재 환자 확보
- **우선순위**: MVP 필수
- **구성**: 하단 고정바 (이름, 연락처, 개인정보 동의, 상담예약 버튼, 빠른상담 전화 버튼)

### FEAT-3: 관리자 콘텐츠 관리
- **목적**: 비개발자가 콘텐츠를 직접 관리
- **Why**: 증례사진, 공지사항 등 주기적 업데이트 필요
- **우선순위**: MVP 필수
- **기능**: 증례사진 CRUD, 증례글 수정, 상담 DB 조회, 공지사항 관리

## 3. 사용자 스토리

### 환자 (방문자)
- As a 치과 환자, I want 진료 분야별 상세 정보를 확인, so that 내원 전 충분히 이해하고 신뢰를 형성할 수 있다
- As a 치과 환자, I want 원장님 약력과 전문성을 확인, so that 안심하고 진료를 맡길 수 있다
- As a 치과 환자, I want 하단바에서 바로 상담 신청, so that 편하게 연락처를 남길 수 있다
- As a 치과 환자, I want 빠른상담 버튼으로 전화 연결, so that 즉시 상담을 받을 수 있다
- As a 소아 보호자, I want 소아치과 정보와 웃음가스 설명을 확인, so that 아이 치료를 안심하고 결정할 수 있다

### 관리자
- As a 관리자, I want 증례사진을 업로드/수정, so that 최신 치료 사례를 보여줄 수 있다
- As a 관리자, I want 상담 신청 DB를 확인, so that 환자에게 빠르게 연락할 수 있다
- As a 관리자, I want 공지사항(휴진일정 등)을 관리, so that 환자에게 정확한 정보를 전달할 수 있다
